model_inference.py

import joblib

import pandas as pd


# Load trained pipeline

model = joblib.load("parking_prob_model.pkl")


def predict_occupancy(hour, day):

features = pd.DataFrame([[hour, day]])

probability = model.predict_proba(features)

return probability

Predictive Intelligence

  • Temporal Analysis

    Model analyzes historical occupancy rates based on time-of-day and day-of-week.

  • Probabilistic Output

    Instead of binary (Full/Empty), we provide probability scores (e.g., "85% chance of finding a spot").

  • Continuous Learning

    System retrains periodically on new sensor data to adapt to changing traffic patterns.
