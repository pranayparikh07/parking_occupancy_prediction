01. ACQUISITION

Signal Detection

IR sensors detect vehicle presence. Arduino Mega processes the digital signal and packages it for transmission.

02. INGESTION

Server Uplink

ESP8266 connects to WiFi and sends a POST request to the Flask endpoint. Data is logged to a persistent CSV file.

03. LEARNING

Model Training

The CSV dataset is used to retrain the ML model, learning occupancy patterns based on time and day.

04. INSIGHT

Dashboard

Users view real-time status and predictive availability on the web dashboard, powered by the trained model.
